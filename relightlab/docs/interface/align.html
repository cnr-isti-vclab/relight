Align

RTI and normal computations require all the images to be taken from the same view-point. All the images needs to be perfectly aligned for good results: ghosting or blurring will be generated otherwise.

You can verify, and eventually correct the alignment of the images bu first create a new alignment by clickeing the butt9on and then select a small rectangle clicking on the image.

The application with collect the same patch from all the images and you can inspect the alignment using the verify button. You might also change the alignent of an image dragging the cross and make sure that cross has the same position relative to the content of the thumbnails.

You might want to carefully select the patches: the best regions usually are flat regions with some texture details, placing a small printed marker on the edge of the image before capturing the dataset is advised and a perfect target for verifying the alignment. In general avoid regions with shadows or reflections.
