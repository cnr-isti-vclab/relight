New Project

A dialog will ask you to select a folder containing the images to be processed.

All the images must be in the same format and at the same resolution. The following image formats are supported:

  • JPEG (.jpg, .jpeg) — 8-bit, up to 4 channels
  • PNG (.png) — 8-bit and 16-bit, up to 4 channels
  • TIFF (.tif, .tiff) — 8-bit, 16-bit, and 32-bit float, up to 4 channels
  • OpenEXR (.exr) — 32-bit float, multi-channel
