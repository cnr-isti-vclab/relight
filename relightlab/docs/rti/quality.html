RTI quality

JPEG Quality

The JPEG quality setting controls the compression level of the image planes. Higher values (e.g., 95) preserve more detail but result in larger file sizes, while lower values (e.g., 75) reduce file size at the cost of visible compression artifacts. For most applications, a quality of 95 is recommended to minimize loss of fine details in the RTI fitting coefficients.

Output Colorspace

The output colorspace determines both the working space used during RTI computation and the colorspace in which the final RTI planes are stored and displayed.

sRGB (Default)

sRGB is the standard colorspace for most web displays and image viewers. When selected:

  • Input images are converted to sRGB
  • RTI coefficients are computed in sRGB space
  • Planes are stored in sRGB
  • Provides better detail in shadows and dark regions due to the gamma correction applied by the sRGB curve
  • Least squares fitting error is weighted to emphasize darker values, improving shadow accuracy

Linear RGB

Linear RGB represents light intensity without gamma correction. When selected:

  • Input images are converted to linear RGB
  • RTI coefficients are computed in linear RGB space
  • Planes are stored in linear RGB
  • Provides more faithful representation of highlights with reduced posterization
  • More accurate for applications requiring physical accuracy (e.g., normal maps, shape-from-shading)
  • PTM planes computed in linear space are more suitable for deriving normals or surface shape

Choosing Between sRGB and Linear

The choice depends on your use case and subject matter:

  • Use sRGB for general archaeological or cultural heritage objects with significant shadow detail that is important for visual analysis
  • Use Linear for scientific applications, shape recovery, normal map generation, or objects where highlight detail is critical
  • The gamma curve in sRGB (and its perceptual weighting) produces more visually pleasing results for typical viewing, but may lose subtle highlight gradations
  • Linear space preserves physical accuracy but may show more banding in shadows unless sufficient JPEG quality is maintained
